# Real-Time Collaborative Quiz

## Tech Stack

- Next.js + TypeScript
- Socket.IO

## Features

- Global question sync across multiple users.
- Real-time leaderboard updates.
- Individual feedback for incorrect answers.
- Collaborative quiz progression.

## Dependencies

- `next`: 14.2.14
- `react`: ^18
- `react-dom`: ^18
- `socket.io`: ^4.8.0
- `socket.io-client`: ^4.8.0
